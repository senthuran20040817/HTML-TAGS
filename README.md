# HTML Tags 🏷️

A personal reference repository documenting my journey through the HTML module — covering the tags, attributes, and structures that form the foundation of web development.

## 📚 What's Inside

This repository contains notes, examples, and practice files covering:

- **Document structure** — `<!DOCTYPE>`, `<html>`, `<head>`, `<body>`
- **Text & headings** — `<h1>`–`<h6>`, `<p>`, `<span>`, `<strong>`, `<em>`
- **Links & media** — `<a>`, `<img>`, `<video>`, `<audio>`
- **Lists** — `<ul>`, `<ol>`, `<li>`, `<dl>`
- **Tables** — `<table>`, `<tr>`, `<th>`, `<td>`
- **Forms** — `<form>`, `<input>`, `<label>`, `<select>`, `<button>`
- **Semantic HTML** — `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`
- **Inline vs block elements** and when to use each

## 🎯 Purpose

This repo serves as my personal HTML reference and revision material. Each file is a hands-on example I built while learning, so I can revisit concepts and track my progress over time.

## 🛠️ How to Use

Clone the repository and open any `.html` file directly in a browser — no setup needed.

```bash
git clone https://github.com/your-username/html-tags.git
```
